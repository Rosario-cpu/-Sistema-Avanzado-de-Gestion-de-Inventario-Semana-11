Sistema de Gestión de Inventario - Semana 11

Este proyecto es una aplicación de consola desarrollada en Python para la administración eficiente de un inventario de productos.

Características Técnicas:
Estructuras Compuestas: Se utiliza un Diccionario como estructura principal para almacenar el conglomerado de datos, permitiendo un acceso rápido y organizado.

Persistencia de Datos: El sistema implementa la lectura y escritura en un archivo .txt (inventario_datos.txt), asegurando que la información se mantenga de forma permanente.

Programación Orientada a Objetos (POO): Se integran clases y métodos para gestionar las operaciones del inventario, siguiendo un modelado del mundo real.

Asociaciones Dinámicas: Se establecen vínculos dinámicos entre claves únicas (IDs) y objetos de la clase Producto.
